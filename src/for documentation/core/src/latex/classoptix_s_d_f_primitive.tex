\doxysubsection{Detailed Description}
A basic class for all sdf primitives. 

The class creates a basic triangular plane for Optix use. Codes from Optix advanced samples. Architecture much inspired by V\+TK ({\texttt{ http\+://vtk.\+org}}) implementation
