This tutorial demonstrates basics of geometry creation according to {\texttt{ F\+Rep modelling approach}}. At the same time tutorial demonstrates the basics of A\+PI use and main concept that is much simillar to {\texttt{ Visualisation Toolkit}}. For similarities and differences with V\+TK see \doxyref{Main logic and similarities with V\+TK A\+PI}{p.}{md_doc_vtkstyle}\doxysection{F\+Rep primitive creation and rendering}\label{md_doc_freptutorial_autotoc_md5}
