Detailed Description

template<typename T>
class optixTextureReader< T >

Basic class for reading texture from file.

Read texture from file and applies signed distance transfrom to generate SDF texture.

Architecture much inspired by VTK (http://vtk.org) implementation

Uses itk signed distance transform
