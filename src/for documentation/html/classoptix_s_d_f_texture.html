Detailed Description

Generates a dynamic SDF primitive on base of sdf textures.

The sdf textures can be obtained on base of signed distance transform procedure. In the API the ITK signed distance transform is used to obtain such textures. See optixSDFTextureReader class: optixSDFTextureReader::optixSDFTextureReader() for example.
