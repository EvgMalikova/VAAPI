Detailed Description

"per ray data" (PRD) for our sample's rays. In the simple example, there is only one ray type, and it only ever returns one thing, which is a color (everything else is handled through the recursion). In addition to that return type, rays have to carry recursion state, which in this case are recursion depth and random number state
