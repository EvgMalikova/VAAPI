Detailed Description

Architecture much inspired by VTK Renderer(http://vtk.org) implementation The classes are mainly responsible for openGL/openAL interop.

The class responsible for rendering auditory propertis.

The model is used by vaRenderer to render auditory image of data The class is mainly responsible for cuda/openAL interop
