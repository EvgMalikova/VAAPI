Detailed Description

Abstract class for SDF Geometry. A basic class. All SDF geometry inherits it's functions.

All SDF geometry inherits it's functions, that includes:

1)SDF dynamic textures

2)FRep modelling primitives and operations that construct dynamically Construction Tree see http://hyperfun.org for details

3)Large objects, like molecular structures, that consist of lot's of objects and use BVH for rendering

...

may be something else

Architecture much inspired by VTK (http://vtk.org) implementation. Main operations similar to most of objects are much similar to VTK classes structure:

virtual void SetContext(optix::Context &context);

virtual void Update();

virtual optix::Geometry GetOutput()

virtual sdfGeo* GetOutputDesc()

See Tutorial for more info.
