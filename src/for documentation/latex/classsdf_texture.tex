\doxysubsection{Detailed Description}
Generates a dynamic S\+DF primitive on base of sdf textures. 

The sdf textures can be obtained on base of signed distance transform procedure. In the A\+PI the I\+TK signed distance transform is used to obtain such textures. See \mbox{\hyperlink{classsdf_texture_reader}{sdf\+Texture\+Reader}} class\+: \mbox{\hyperlink{classsdf_texture_reader_aca937f77e8cb7231de6e6a60823ba1ae}{sdf\+Texture\+Reader\+::\+Preprocess\+Data()}} function for example.
