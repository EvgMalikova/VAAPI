\subsubsection*{template$<$typename T$>$\newline
class optix\+Texture\+Reader$<$ T $>$}

Basic class for reading texture from file. 

Read texture from file and applies signed distance transfrom to generate S\+DF texture.

Architecture much inspired by V\+TK (\href{http://vtk.org}{\texttt{ http\+://vtk.\+org}}) implementation
