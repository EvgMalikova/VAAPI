\doxysubsection{Detailed Description}
Generates a dynamic S\+DF primitive on base of sdf textures. 

The sdf textures can be obtained on base of signed distance transform procedure. In the A\+PI the I\+TK signed distance transform is used to obtain such textures. See \mbox{\hyperlink{classoptix_s_d_f_texture_reader}{optix\+S\+D\+F\+Texture\+Reader}} class\+: \mbox{\hyperlink{classoptix_s_d_f_texture_reader_a1bb8c59f4ddbff1515a2b37874b74cbc}{optix\+S\+D\+F\+Texture\+Reader\+::optix\+S\+D\+F\+Texture\+Reader()}} for example.
