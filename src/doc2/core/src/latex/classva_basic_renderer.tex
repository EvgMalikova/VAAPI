\doxysubsection{Detailed Description}
Abstract visual-\/auditory renderer. 

Abstract visual-\/auditory renderer. The class is responsible for\+:
\begin{DoxyItemize}
\item setting optical model
\item setting auditory model
\item specifies main rendering modes in A\+PI enum Render\+Modes
\item launches ray-\/tracing of visual and optical stimuli 
\end{DoxyItemize}
