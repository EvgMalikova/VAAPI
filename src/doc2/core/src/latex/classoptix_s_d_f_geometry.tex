\doxysubsection{Detailed Description}
Abstract class for S\+DF Geometry. A basic class. All S\+DF geometry inherits it\textquotesingle{}s functions. 

All S\+DF geometry inherits it\textquotesingle{}s functions, that includes\+:

1)S\+DF dynamic textures

2)F\+Rep modelling primitives and operations that construct dynamically Construction Tree see {\texttt{ http\+://hyperfun.\+org}} for details

3)Large objects, like molecular structures, that consist of lot\textquotesingle{}s of objects and use B\+VH for rendering

...

may be something else

Architecture much inspired by V\+TK ({\texttt{ http\+://vtk.\+org}}) implementation. Main operations similar to most of objects are much similar to V\+TK classes structure\+:
